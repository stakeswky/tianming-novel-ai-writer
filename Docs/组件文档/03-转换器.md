# 🔄 转换器

[← 返回索引](./组件总索引.md)

本文档包含XAML数据绑定转换器（13个）。

所有转换器已在 `Converters.xaml` 中注册为全局资源，可直接使用 `{StaticResource ConverterName}`。

---

## 可见性转换器

**文件位置**: `Framework/Common/Converters/Visibility/`

| 转换器 | 用途 | 使用方式 |
|--------|------|---------|
| BoolToVisibilityConverter | bool → Visibility | `{StaticResource BoolToVisibilityConverter}` |
| InverseBoolToVisibilityConverter | !bool → Visibility | `{StaticResource InverseBoolToVisibilityConverter}` |
| NullToVisibilityConverter | null → Collapsed | `{StaticResource NullToVisibilityConverter}` |
| InverseNullToVisibilityConverter | null → Visible | `{StaticResource InverseNullToVisibilityConverter}` |
| CountToVisibilityConverter | count>0 → Visible | `{StaticResource CountToVisibilityConverter}` |
| InverseCountToVisibilityConverter | count==0 → Visible | `{StaticResource InverseCountToVisibilityConverter}` |

### 使用示例

```xml
<!-- bool转Visibility -->
<TextBlock Text="已启用" 
           Visibility="{Binding IsEnabled, Converter={StaticResource BoolToVisibilityConverter}}"/>

<!-- 反转bool -->
<TextBlock Text="已禁用" 
           Visibility="{Binding IsEnabled, Converter={StaticResource InverseBoolToVisibilityConverter}}"/>

<!-- null判断 -->
<TextBlock Text="无数据" 
           Visibility="{Binding Data, Converter={StaticResource NullToVisibilityConverter}}"/>

<!-- 数量判断 -->
<TextBlock Text="暂无数据" 
           Visibility="{Binding Items.Count, Converter={StaticResource InverseCountToVisibilityConverter}}"/>
```

## 值转换器

**文件位置**: `Framework/Common/Converters/Value/`

| 转换器 | 用途 | 文件 |
|--------|------|------|
| InverseBoolConverter | bool反转 | InverseBoolConverter.cs |
| EnumToBoolConverter | 枚举转bool | EnumToBoolConverter.cs |
| BoolToTextConverter | bool转文本 | BoolToTextConverter.cs |
| PercentageToWidthConverter | 百分比→宽度（MultiBinding） | PercentageToWidthConverter.cs |
| ResourceKeyToDynamicResourceConverter | 资源键→主题画刷 | ResourceKeyToDynamicResourceConverter.cs |

### 使用示例

```xml
<!-- bool反转 -->
<CheckBox IsChecked="{Binding IsDisabled, Converter={StaticResource InverseBoolConverter}}"/>

<!-- 枚举转bool（用于RadioButton） -->
<RadioButton IsChecked="{Binding CurrentMode, Converter={StaticResource EnumToBoolConverter}, ConverterParameter=Edit}"/>

<!-- 百分比转宽度（用于进度条宽度等） -->
<Grid>
    <Grid.ColumnDefinitions>
        <ColumnDefinition Width="Auto" />
        <ColumnDefinition Width="*" />
    </Grid.ColumnDefinitions>

    <Border Grid.Column="0" Width="100" Background="LightGray"/>

    <Border Grid.Column="1" Background="DodgerBlue">
        <Border.Width>
            <MultiBinding Converter="{StaticResource PercentageToWidthConverter}">
                <!-- 百分比（0-1） -->
                <Binding Path="ProgressRatio"/>
                <!-- 容器实际宽度 -->
                <Binding RelativeSource="{RelativeSource AncestorType=Border}" Path="ActualWidth"/>
            </MultiBinding>
        </Border.Width>
    </Border>
</Grid>

<!-- 资源键转主题画刷（支持 Color 或 SolidColorBrush 资源） -->
<Border Background="{Binding ThemeColorKey, Converter={StaticResource ResourceKeyToDynamicResourceConverter}}"/>
```

## 比较转换器

**文件位置**: `Framework/Common/Converters/Comparison/`

| 转换器 | 用途 | 文件 |
|--------|------|------|
| ObjectEqualityConverter | 对象相等性比较 | ObjectEqualityConverter.cs |

### 使用示例

```xml
<!-- 对象相等性比较（用于选中状态） -->
<Border Background="{DynamicResource PrimaryColor}">
    <Border.Style>
        <Style TargetType="Border">
            <Setter Property="Visibility" Value="Collapsed"/>
            <Style.Triggers>
                <DataTrigger Value="True">
                    <DataTrigger.Binding>
                        <MultiBinding Converter="{StaticResource ObjectEqualityConverter}">
                            <Binding Path="SelectedItem"/>
                            <Binding Path="."/>
                        </MultiBinding>
                    </DataTrigger.Binding>
                    <Setter Property="Visibility" Value="Visible"/>
                </DataTrigger>
            </Style.Triggers>
        </Style>
    </Border.Style>
</Border>
```

## 布局转换器

**文件位置**: `Framework/Common/Converters/LevelToMarginConverter.cs`

| 转换器 | 用途 | 使用方式 |
|--------|------|---------|
| LevelToMarginConverter | 根据层级计算左侧缩进边距 | `{StaticResource LevelToMarginConverter}` |

### 功能说明

- **用途**: 自动将节点的 `Level` 属性转换为左侧缩进边距（Margin）
- **默认缩进**: 每级20px
- **应用场景**: 树形数据展示（DataTreeView、TreeComboBox）

### 属性配置

| 属性 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| IndentPerLevel | double | 20.0 | 每级缩进像素 |
| BaseLeftMargin | double | 0.0 | 基础左边距 |

### 使用示例

```xml
<!-- 资源注册（已在 Converters.xaml 中注册） -->
<converters:LevelToMarginConverter x:Key="LevelToMarginConverter" IndentPerLevel="20"/>

<!-- 在 DataTemplate 中使用 -->
<DataTemplate>
    <Button Margin="{Binding Level, Converter={StaticResource LevelToMarginConverter}}">
        <StackPanel Orientation="Horizontal">
            <emoji:TextBlock Text="{Binding Icon}" FontSize="14" Margin="0,0,5,0"/>
            <TextBlock Text="{Binding Name}"/>
        </StackPanel>
    </Button>
</DataTemplate>
```

### 转换逻辑

```csharp
// Level 1: Margin = (0, 2, 0, 2)     // 顶级不缩进
// Level 2: Margin = (20, 2, 0, 2)    // 缩进 20px
// Level 3: Margin = (40, 2, 0, 2)    // 缩进 40px
// Level 4: Margin = (60, 2, 0, 2)    // 缩进 60px
// Level 5: Margin = (80, 2, 0, 2)    // 缩进 80px
```

### 配套使用

- **DataTreeView**: 自动应用此转换器实现5层递归视觉缩进
- **TreeComboBoxStyle**: 父分类下拉框的层级缩进
- **TreeNodeItem**: 数据模型需包含 `Level` 属性（int类型）
