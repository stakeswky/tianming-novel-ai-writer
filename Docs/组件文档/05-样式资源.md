</ResourceDictionary.MergedDictionaries>

## 输入框样式

**文件位置**: `Framework/Common/Styles/InputStyles/`

| 样式名 | 用途 | 使用方式 |
|--------|------|---------|
| StandardTextBoxStyle | 标准文本输入框 | `Style="{StaticResource StandardTextBoxStyle}"` |
| StandardComboBoxStyle | 标准下拉框 | `Style="{StaticResource StandardComboBoxStyle}"` |
| StandardPasswordBoxStyle | 标准密码框 | `Style="{StaticResource StandardPasswordBoxStyle}"` |
| **TreeComboBoxStyle** | **树形下拉框（5层递归）v2.3新增** | `Style="{StaticResource TreeComboBoxStyle}"` |

### 基础示例

```xml
<TextBox Text="{Binding Name}" 
         Style="{StaticResource StandardTextBoxStyle}" 
         Height="36"/>

<ComboBox ItemsSource="{Binding Items}" 
          Style="{StaticResource StandardComboBoxStyle}" 
          Height="36"/>
```

### 统一文本编辑右键菜单（v4.1 增强）

自 v4.1 起，输入框相关的**隐式样式**会自动为 `TextBox` 和 `PasswordBox` 启用标准的 3 项编辑菜单：

- 剪切(T)
- 复制(C)
- 粘贴(V)

实现方式（节选自 `ImplicitStyles.xaml`）：

```xml
<Style TargetType="TextBox" BasedOn="{StaticResource StandardTextBoxStyle}">
    <Setter Property="helpers:TextInputContextMenuHelper.EnableStandardEditMenu" Value="True"/>
</Style>

<Style TargetType="PasswordBox" BasedOn="{StaticResource StandardPasswordBoxStyle}">
    <Setter Property="helpers:TextInputContextMenuHelper.EnableStandardEditMenu" Value="True"/>
</Style>
```

> ✅ 只要引用了 `AllStyles.xaml`，并且未自定义 ContextMenu，所有文本输入控件（包括密码框）都会自动获得统一的现代圆角右键菜单，无需在业务 XAML 手动配置。

### TreeComboBoxStyle（v2.3新增）

**文件位置**: `Framework/Common/Styles/InputStyles/TreeComboBoxStyle.xaml`

专为多层级分类选择设计的树形下拉框样式。

#### 功能特性
- ✅ 支持5层递归显示
- ✅ 自动显示"主页导航"顶级选项
- ✅ 单击展开/折叠，双击选择
- ✅ 显示完整路径（如 "角色 > 主角 > 属性"）
- ✅ 集成DataTreeView组件
- ✅ 自动层级缩进（每级20px）
- ✅ **下拉框宽度同步**（v4.1更新）：下拉弹窗宽度精确同步ComboBox本身宽度，视觉统一

#### 使用示例

```xml
<!-- 基础用法（内容模块） -->
<ComboBox Style="{StaticResource TreeComboBoxStyle}"
          ItemsSource="{Binding CategorySelectionTree}"
          IsDropDownOpen="{Binding IsCategoryTreeDropdownOpen, Mode=TwoWay}"
          helpers:ComboBoxHelper.SelectedPath="{Binding SelectedCategoryTreePath, Mode=TwoWay}"
          helpers:ComboBoxHelper.DisplayIcon="{Binding SelectedCategoryTreeIcon, Mode=TwoWay}"
          helpers:ComboBoxHelper.MaxLevel="5"
          helpers:ComboBoxHelper.ShowTopLevelOption="True"
          helpers:ComboBoxHelper.NodeDoubleClickCommand="{Binding CategoryTreeNodeSelectCommand}" />

<!-- 完整用法（分类管理标准方案） -->
<ComboBox Style="{StaticResource TreeComboBoxStyle}"
          ItemsSource="{Binding ParentCategoryTree}"
          IsDropDownOpen="{Binding IsTreeComboBoxOpen, Mode=TwoWay}"
          helpers:ComboBoxHelper.SelectedPath="{Binding SelectedParentPath, Mode=TwoWay}"
          helpers:ComboBoxHelper.DisplayIcon="{Binding SelectedParentIcon, Mode=TwoWay}"
          helpers:ComboBoxHelper.MaxLevel="4"
          helpers:ComboBoxHelper.NodeDoubleClickCommand="{Binding SelectParentCategoryCommand}" />
```

#### 配套使用
- **数据源**: `ObservableCollection<TreeNodeItem>`
- **辅助类**: `ComboBoxHelper.SelectedPath`、`ComboBoxHelper.MaxLevel`、`ComboBoxHelper.NodeDoubleClickCommand`
- **转换器**: `LevelToMarginConverter`（自动应用）

## ListBox样式

**文件位置**: `Framework/Common/Styles/ListBoxStyles/StandardListBox.xaml`

### 隐式样式（自动应用）

**CardListBoxStyle** - 卡片式ListBox选中效果

- 未选中：灰色细边框
- 悬停：浅蓝背景
- 选中：蓝色粗边框

```xml
<!-- 直接使用ListBox即可，自动应用CardListBoxStyle -->
<ListBox ItemsSource="{Binding Items}">
    <ListBox.ItemTemplate>
        <DataTemplate>
            <TextBlock Text="{Binding Name}"/>
        </DataTemplate>
    </ListBox.ItemTemplate>
</ListBox>
```

### 显式样式

| 样式名 | 用途 |
|--------|------|
| CompactCardListBoxStyle | 紧凑版卡片样式 |
| UltraCompactCardListBoxStyle | 超紧凑版卡片样式 |

```xml
<ListBox Style="{StaticResource CompactCardListBoxStyle}" 
         ItemsSource="{Binding Items}"/>
```

## 弹窗样式（v4.2新增）

**文件位置**: `Framework/Common/Styles/DialogStyles/StandardDialogStyles.xaml`

提供标准弹窗的可复用样式组件，完全兼容 x:Name。

| 样式名 | 类型 | 用途 |
|--------|------|------|
| StandardDialogBorderStyle | Border | 弹窗外框（白色背景、圆角12、边框） |
| StandardDialogTitleBarStyle | Border | 标题栏（浅灰背景 #F9FAFB、固定高度44px） |
| StandardDialogCloseButtonStyle | Button | 关闭按钮（红色悬停效果） |
| StandardDialogTitleTextStyle | TextBlock | 标题文本（14px、SemiBold） |
| StandardDialogFooterStyle | StackPanel | 底部按钮区（右对齐） |

### Window 属性说明

以下属性必须在 Window 声明中直接设置（不能通过 Style）：

```xml
WindowStyle="None"
AllowsTransparency="True"
Background="Transparent"
ShowInTaskbar="False"
ResizeMode="CanResize"
WindowStartupLocation="CenterOwner"
```

### 使用示例

```xml
<Window ...
        WindowStyle="None"
        AllowsTransparency="True"
        Background="Transparent"
        ResizeMode="CanResize">
    
    <WindowChrome.WindowChrome>
        <WindowChrome CaptionHeight="44" CornerRadius="12" ResizeBorderThickness="6"/>
    </WindowChrome.WindowChrome>
    
    <Border Style="{DynamicResource StandardDialogBorderStyle}">
        <Grid>
            <!-- 标题栏 -->
            <Border Style="{DynamicResource StandardDialogTitleBarStyle}">
                <Grid>
                    <StackPanel Orientation="Horizontal" Margin="16,0,0,0">
                        <emoji:TextBlock Text="📋" FontSize="16" Margin="0,0,8,0"/>
                        <TextBlock Text="标题" Style="{DynamicResource StandardDialogTitleTextStyle}"/>
                    </StackPanel>
                    <Button Grid.Column="1" 
                            Click="Close_Click" 
                            Style="{DynamicResource StandardDialogCloseButtonStyle}"/>
                </Grid>
            </Border>
            
            <!-- 内容区 - 可使用 x:Name -->
            <Grid Grid.Row="1">
                <TextBox x:Name="MyTextBox"/>  <!-- 完全兼容 x:Name -->
            </Grid>
            
            <!-- 底部按钮 -->
            <StackPanel Grid.Row="2" Style="{DynamicResource StandardDialogFooterStyle}">
                <Button Content="确定" Style="{DynamicResource PrimaryButtonStyle}"/>
            </StackPanel>
        </Grid>
    </Border>
</Window>
```

## UI一致性规范

### 圆角标准
- 控件圆角：6px（Button、TextBox、ComboBox）
- 容器圆角：8px（GroupBox、Card、Panel）

### 间距标准
- 紧密间距：5px
- 标准间距：10px
- 大间距：20px
- 区域分隔：30px

### 按钮内边距
- 标准按钮：`Padding="12,6"`
- 输入框：`Padding="12,0"`
