# 🎭 行为类

[← 返回索引](./组件总索引.md)

本文档包含WPF附加行为类（2个）。

---

## 1. FocusBehavior - 焦点管理

**文件位置**: `Framework/Common/Behaviors/FocusBehavior.cs`

控制控件焦点。

### 使用示例

```xml
xmlns:behaviors="clr-namespace:TM.Framework.Common.Behaviors"

<TextBox behaviors:FocusBehavior.IsFocused="{Binding ShouldFocus}"/>
```

### 应用场景

- 对话框打开时自动聚焦到输入框
- 表单验证失败后聚焦到错误字段
- 搜索框自动聚焦

---

## 2. SmoothScrollBehavior - 平滑滚动

**文件位置**: `Framework/Common/Behaviors/SmoothScrollBehavior.cs`

为ScrollViewer添加平滑滚动效果。

### 使用示例

```xml
<ScrollViewer behaviors:SmoothScrollBehavior.IsEnabled="True">
    <StackPanel>
        <!-- 内容 -->
    </StackPanel>
</ScrollViewer>
```

### 应用场景

- 长列表滚动优化
- 内容区域平滑滚动
- 提升用户体验
