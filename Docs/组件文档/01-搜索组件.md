## 1. SearchTextBox - 标准搜索框

**文件位置**: `Framework/Common/Search/SearchTextBox.xaml`

标准化搜索输入框，提供统一的搜索体验。

### 功能特性
- ✅ 彩色emoji图标占位符（🔍）
- ✅ 实时双向绑定
- ✅ 自定义占位符文本
- ✅ 统一36px高度
- ✅ 自动显示/隐藏占位符
- ✅ 内置 3 项文本编辑右键菜单（剪切 / 复制 / 粘贴），与右栏输入框风格一致

### 使用示例

```xml
<!-- XAML -->
xmlns:search="clr-namespace:TM.Framework.Common.Search"

<search:SearchTextBox SearchKeyword="{Binding SearchKeyword}"
                     PlaceholderText="搜索标题、内容、标签..."/>
```

```csharp
// ViewModel
private string _searchKeyword = string.Empty;

public string SearchKeyword
{
    get => _searchKeyword;
    set
    {
        if (_searchKeyword != value)
        {
            _searchKeyword = value;
            OnPropertyChanged();
            PerformSearch(); // 实时搜索
        }
    }
}
```

### 依赖属性

| 属性 | 类型 | 默认值 | 说明 |
|-----|------|--------|------|
| SearchKeyword | string | "" | 搜索关键词（双向绑定） |
| PlaceholderText | string | "搜索..." | 占位符提示文本 |

## 2. SearchHelper - 智能搜索工具类

**文件位置**: `Framework/Common/Search/SearchHelper.cs`

提供统一的智能搜索算法和评分系统。

### 功能特性
- ✅ 10级智能评分系统
- ✅ 多字段匹配支持
- ✅ 自动按相关度排序
- ✅ 泛型方法支持任意类型

### 评分规则

| 匹配类型 | 分数 | 说明 |
|---------|------|------|
| 完全匹配 | 10000 | 目标 == 关键词 |
| 开头匹配 | 5000 | 以关键词开头 |
| 包含匹配 | 2000+ | 包含关键词 |
| 首字母匹配 | 500 | 首字母组合匹配 |
| 额外字段匹配 | 400-800 | 额外字段匹配 |

### 使用示例

```csharp
using TM.Framework.Common.Search;

// 方法1：智能搜索并排序（推荐）
var results = SearchHelper.FilterAndSort(
    items: allBlocks,
    keyword: SearchKeyword,
    getTargetText: block => block.Title,  // 主字段
    getAdditionalFields: block => new[] {  // 额外字段
        block.Content, 
        string.Join(" ", block.Tags)
    }
);

// 方法2：仅计算匹配分数
int score = SearchHelper.CalculateMatchScore(
    targetText: "GPT-4 Turbo",
    keyword: "gpt",
    additionalFields: new[] { "OpenAI", "chat" }
);
```

### API参考

```csharp
// 智能搜索并排序
public static List<T> FilterAndSort<T>(
    IEnumerable<T> items,
    string keyword,
    Func<T, string> getTargetText,
    Func<T, string[]> getAdditionalFields = null
)

// 计算匹配分数
public static int CalculateMatchScore(
    string targetText,
    string keyword,
    params string[] additionalFields
)
```

### 应用场景

#### 场景1：知识块搜索
```csharp

    allBlocks,
    SearchKeyword,
    block => block.Title,
    block => new[] { block.Content, string.Join(" ", block.Tags) }

```

#### 场景2：模板搜索


#### 场景3：模型搜索
```csharp

    SearchKeyword,

    model => new[] { model.Name, model.ProviderName }

```

## 📝 使用规范

### 必须使用标准搜索组件

所有模块的搜索功能必须使用 `SearchTextBox` + `SearchHelper` 实现，禁止自定义搜索框。

> ℹ️ **说明**：`SearchTextBox` 内部已集成 `TextInputContextMenuHelper`，默认带有标准 3 项编辑右键菜单，无需额外为搜索输入框实现 ContextMenu。

### 搜索性能优化

1. **大数据集优化**
   - 使用虚拟化（VirtualizingStackPanel）
   - 避免在UI线程执行耗时搜索

2. **实时搜索优化**
   - 在`SearchKeyword`属性setter中触发搜索
   - 可添加防抖逻辑避免频繁触发

3. **空关键词处理**
   - `SearchHelper.FilterAndSort()` 空关键词时返回原始列表
   - `CalculateMatchScore()` 空关键词时返回1000分
