## 一、问题描述

使用 `TwoColumnEditorLayout` 的业务模块存在以下问题：
1. **布局紧凑、底部留白** - 内容未填满可用空间
2. **重复标题** - `IconHeaderGroupBox` 内部又有 `TextBlock` 标题
3. **双边框** - `TextBox` 边框与 `IconHeaderGroupBox` 边框重叠
4. **Tab标题未定义** - 显示默认的 "Tab3" 等

---

## 二、修改规范

### 2.1 Tab1（EditAdditionalContent）修改

**问题**: `StackPanel` 无法让内容填满垂直空间

**修改前**:
```xml
<dm:TwoColumnEditorLayout.EditAdditionalContent>
    <StackPanel>
        <layout:IconHeaderGroupBox Icon="📋" Title="基础信息" Margin="0,0,0,10">
            ...
        </layout:IconHeaderGroupBox>
        <layout:IconHeaderGroupBox Icon="✅" Title="校验结果" Margin="0,0,0,0">
            ...
        </layout:IconHeaderGroupBox>
    </StackPanel>
</dm:TwoColumnEditorLayout.EditAdditionalContent>
```

**修改后**:
```xml
<dm:TwoColumnEditorLayout.EditAdditionalContent>
    <Grid>
        <Grid.RowDefinitions>
            <RowDefinition Height="Auto"/>
            <RowDefinition Height="*"/>
        </Grid.RowDefinitions>
        <layout:IconHeaderGroupBox Grid.Row="0" Icon="📋" Title="基础信息" Margin="0,0,0,10">
            ...
        </layout:IconHeaderGroupBox>
        <layout:IconHeaderGroupBox Grid.Row="1" Icon="✅" Title="校验结果" Margin="0">
            ...
        </layout:IconHeaderGroupBox>
    </Grid>
</dm:TwoColumnEditorLayout.EditAdditionalContent>
```

**关键点**:
- `StackPanel` → `Grid`
- 最后一个 `RowDefinition` 设为 `Height="*"` 填满剩余空间
- 最后一个 `IconHeaderGroupBox` 的 `Margin` 改为 `"0"`

#### 多GroupBox场景（3个以上）

```xml
<Grid>
    <Grid.RowDefinitions>
        <RowDefinition Height="Auto"/>  <!-- 第1个 -->
        <RowDefinition Height="Auto"/>  <!-- 第2个 -->
        <RowDefinition Height="*"/>     <!-- 最后一个填满 -->
    </Grid.RowDefinitions>
    <layout:IconHeaderGroupBox Grid.Row="0" ... Margin="0,0,0,10"/>
    <layout:IconHeaderGroupBox Grid.Row="1" ... Margin="0,0,0,10"/>
    <layout:IconHeaderGroupBox Grid.Row="2" ... Margin="0"/>
</Grid>
```

### 2.2 Tab2/3/4（DetailsContent/Tab3Content/Tab4Content）修改

#### 单字段Tab（TextBox填满整个Tab）

**修改前**:
```xml
<dm:TwoColumnEditorLayout.DetailsContent>
    <ScrollViewer VerticalScrollBarVisibility="Auto">
        <layout:IconHeaderGroupBox Icon="📄" Title="详细内容" Margin="0">
            <StackPanel>
                <TextBlock Text="字段标题" Margin="0,0,0,5"/>
                <TextBox Text="{Binding FormField}" MinHeight="200"
                         Style="{DynamicResource StandardTextBoxStyle}"/>
            </StackPanel>
        </layout:IconHeaderGroupBox>
    </ScrollViewer>
</dm:TwoColumnEditorLayout.DetailsContent>
```

**修改后**:
```xml
<dm:TwoColumnEditorLayout.DetailsContent>
    <layout:IconHeaderGroupBox Icon="📄" Title="详细内容" Margin="0">
        <TextBox Text="{Binding FormField}"
                 AcceptsReturn="True" TextWrapping="Wrap"
                 VerticalScrollBarVisibility="Auto"
                 Height="Auto" VerticalAlignment="Stretch"
                 BorderThickness="0" Background="Transparent"/>
    </layout:IconHeaderGroupBox>
</dm:TwoColumnEditorLayout.DetailsContent>
```

**关键点**:
- 去掉外层 `ScrollViewer`
- 去掉内层 `StackPanel` 和重复的 `TextBlock` 标题
- `TextBox` 添加: `Height="Auto" VerticalAlignment="Stretch"`
- `TextBox` 添加: `BorderThickness="0" Background="Transparent"` 去双边框

---

#### 多字段Tab（多个TextBox均分高度）

**修改前**:
```xml
<dm:TwoColumnEditorLayout.Tab3Content>
    <ScrollViewer VerticalScrollBarVisibility="Auto">
        <layout:IconHeaderGroupBox Icon="📚" Title="页面标题">
            <StackPanel>
                <TextBlock Text="字段1"/>
                <TextBox Text="{Binding Field1}" MinHeight="100"/>
                <TextBlock Text="字段2"/>
                <TextBox Text="{Binding Field2}" MinHeight="100"/>
            </StackPanel>
        </layout:IconHeaderGroupBox>
    </ScrollViewer>
</dm:TwoColumnEditorLayout.Tab3Content>
```

**修改后**:
```xml
<dm:TwoColumnEditorLayout.Tab3Content>
    <Grid>
        <Grid.RowDefinitions>
            <RowDefinition Height="*"/>
            <RowDefinition Height="*"/>
        </Grid.RowDefinitions>
        <layout:IconHeaderGroupBox Grid.Row="0" Icon="📝" Title="字段1" Margin="0,0,0,10">
            <TextBox Text="{Binding Field1}"
                     AcceptsReturn="True" TextWrapping="Wrap"
                     VerticalScrollBarVisibility="Auto"
                     Height="Auto" VerticalAlignment="Stretch"
                     BorderThickness="0" Background="Transparent"/>
        </layout:IconHeaderGroupBox>
        <layout:IconHeaderGroupBox Grid.Row="1" Icon="📝" Title="字段2" Margin="0">
            <TextBox Text="{Binding Field2}"
                     AcceptsReturn="True" TextWrapping="Wrap"
                     VerticalScrollBarVisibility="Auto"
                     Height="Auto" VerticalAlignment="Stretch"
                     BorderThickness="0" Background="Transparent"/>
        </layout:IconHeaderGroupBox>
    </Grid>
</dm:TwoColumnEditorLayout.Tab3Content>
```

**关键点**:
- 去掉外层 `ScrollViewer` 和页面级 `IconHeaderGroupBox`
- 使用 `Grid` + `RowDefinition Height="*"` 均分高度
- 每个字段独立包裹在 `IconHeaderGroupBox` 中
- 字段标题由 `IconHeaderGroupBox.Title` 承载，去掉内部 `TextBlock`

### 2.4 Border 改为 IconHeaderGroupBox

**修改前**:
```xml
<Border Background="{DynamicResource ContentBackground}"
        BorderBrush="{DynamicResource BorderBrush}"
        BorderThickness="1" CornerRadius="8" Padding="15" Margin="0,0,0,15">
    <StackPanel>
        <StackPanel Orientation="Horizontal" Margin="0,0,0,12">
            <emoji:TextBlock Text="⚙️" FontSize="14"/>
            <TextBlock Text="标题" FontWeight="SemiBold"/>
        </StackPanel>
        <!-- 内容 -->
    </StackPanel>
</Border>
```

**修改后**:
```xml
<layout:IconHeaderGroupBox Icon="⚙️" Title="标题" Margin="0,0,0,10">
    <!-- 内容 -->
</layout:IconHeaderGroupBox>
```

### 2.5 控件类型区分

#### 多行TextBox（需要填满）
```xml
<TextBox Text="{Binding FormField}"
         AcceptsReturn="True" TextWrapping="Wrap"
         VerticalScrollBarVisibility="Auto"
         Height="Auto" VerticalAlignment="Stretch"
         BorderThickness="0" Background="Transparent"/>
```

#### 单行TextBox/ComboBox（保持固定高度）
```xml
<!-- 单行输入框 - 保持原样 -->
<TextBox Text="{Binding FormName}"
         Style="{DynamicResource StandardTextBoxStyle}"
         Height="36"/>

<!-- 下拉框 - 保持原样 -->
<ComboBox ItemsSource="{Binding Options}"
          Style="{DynamicResource StandardComboBoxStyle}"
          Height="36"/>
```

**原则**: 只有多行文本输入（`AcceptsReturn="True"`）才需要填满和去边框

---

### 2.6 保留ScrollViewer的场景

当Tab内容包含**大量参数配置**（如滑块、多个输入组）时，保留ScrollViewer：

```xml
<dm:TwoColumnEditorLayout.DetailsContent>
    <ScrollViewer VerticalScrollBarVisibility="Auto">
        <StackPanel>
            <layout:IconHeaderGroupBox Icon="🎚️" Title="参数调整" Margin="0,0,0,15">
                <!-- 多个Slider、TextBox组合 -->
            </layout:IconHeaderGroupBox>
            <layout:IconHeaderGroupBox Icon="⚙️" Title="高级设置" Margin="0">
                <!-- 更多配置项 -->
            </layout:IconHeaderGroupBox>
        </StackPanel>
    </ScrollViewer>
</dm:TwoColumnEditorLayout.DetailsContent>
```

**适用场景**: ModelManagementView的Tab2（参数调整）、Tab3（全局配置）

---

### 2.7 不需要修改的场景

| 场景 | 原因 |
|------|------|
| `UseStandardScheme2Layout="False"` | 使用自定义布局，不走标准Tab结构 |
| Tab内容为占位页面 | 如 "功能开发中" 提示页 |
| Tab内容为复杂交互组件 | 如对比视图、图表等 |
| 已经使用Grid布局且正常 | 无需重复修改 |

## 三、修改检查清单

### 必做项
- [ ] Tab1: `StackPanel` → `Grid`，最后一个 GroupBox 填满
- [ ] Tab2/3/4 单字段: 去掉 `ScrollViewer`，TextBox 填满
- [ ] Tab2/3/4 多字段: 用 `Grid` 均分高度，每字段独立 GroupBox
- [ ] 多行 TextBox: `Height="Auto" VerticalAlignment="Stretch" BorderThickness="0" Background="Transparent"`
- [ ] Border: 统一改为 `IconHeaderGroupBox`

### 可选项
- [ ] Tab标题: 自定义 `Tab3Header` 等属性（默认显示Tab3/Tab4时需改）
- [ ] 去掉重复 `TextBlock` 标题（标题已在 GroupBox.Title 中）

### 不改项
- [ ] 单行 TextBox/ComboBox: 保持 `Height="36"` 固定高度
- [ ] 参数配置页: 保留 `ScrollViewer`（内容多需滚动）
- [ ] `UseStandardScheme2Layout="False"`: 跳过
- [ ] 占位页面/复杂组件: 跳过
