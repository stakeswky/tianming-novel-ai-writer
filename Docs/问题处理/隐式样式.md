# 隐式样式使用说明

## 📋 什么是隐式样式？

隐式样式（Implicit Styles）是不带 `x:Key` 的样式，会**自动应用**到所有对应类型的控件上。

---

## 🎯 启用方法

### 方式一：全局启用（推荐用于新项目）

编辑 `Framework/Common/Styles/AllStyles.xaml`：

```xml
<!-- 取消注释这一行 -->
<ResourceDictionary Source="ImplicitStyles.xaml"/>
```

### 方式二：局部启用（特定页面）

在需要的页面中引用：

```xaml
<UserControl.Resources>
    <ResourceDictionary>
        <ResourceDictionary.MergedDictionaries>
            <ResourceDictionary Source="pack://application:,,,/Framework/Common/Styles/ImplicitStyles.xaml"/>
        </ResourceDictionary.MergedDictionaries>
    </ResourceDictionary>
</UserControl.Resources>
```

---

## ✅ 启用后的效果

### 之前（需要手动指定）

```xaml
<Button Style="{StaticResource SecondaryButtonStyle}" Content="确定"/>
<TextBlock Style="{StaticResource BodyTextStyle}" Text="内容"/>
<TextBox Style="{StaticResource StandardTextBoxStyle}"/>
```

### 之后（自动应用）

```xaml
<Button Content="确定"/>  <!-- 自动使用 SecondaryButtonStyle -->
<TextBlock Text="内容"/>   <!-- 自动使用 BodyTextStyle -->
<TextBox/>                 <!-- 自动使用 StandardTextBoxStyle -->
```

---

## 🎨 默认样式配置

当前配置的默认样式：

| 控件类型 | 默认样式 | 说明 |
|---------|---------|------|
| `Button` | SecondaryButtonStyle | 次要按钮（边框样式） |
| `TextBlock` | BodyTextStyle | 14px 正文文字 |
| `TextBox` | StandardTextBoxStyle | 标准输入框 |
| `PasswordBox` | StandardPasswordBoxStyle | 标准密码框 |
| `ComboBox` | StandardComboBoxStyle | 标准下拉框 |
| `ListBox` | CardListBoxStyle | 卡片式选中效果（蓝色粗边框+浅蓝悬停） |

---

## 🔄 覆盖默认样式

如需使用其他样式，仍然可以手动指定：

```xaml
<!-- 使用主要按钮样式（覆盖默认） -->
<Button Style="{StaticResource PrimaryButtonStyle}" Content="确定"/>

<!-- 使用标题文字样式（覆盖默认） -->
<TextBlock Style="{StaticResource HeadingStyle}" Text="标题"/>

<!-- 使用危险按钮样式（覆盖默认） -->
<Button Style="{StaticResource DangerButtonStyle}" Content="删除"/>
```

---

## ⚠️ 注意事项

### 1. 影响范围

- ✅ **启用后**：所有新写的控件自动应用样式
- ✅ **未启用**：需要手动指定 `Style="{StaticResource ...}"`

### 2. 现有代码

- 已有的显式样式不受影响
- 只影响**没有指定 Style** 的控件

### 3. 特殊控件

某些特殊控件（如导航按钮、窗口按钮）仍需手动指定专用样式：

```xaml
<!-- 这些仍需手动指定 -->
<Button Style="{StaticResource NavButtonStyle}"/>
<Button Style="{StaticResource SettingsButtonStyle}"/>
<Button Style="{StaticResource WindowButtonStyle}"/>
```

## 🎯 推荐配置

### 新项目/新模块

**建议启用隐式样式**
- 减少重复代码
- 自动保持风格统一
- 提高开发效率

### 现有项目

**建议不启用（或逐步启用）**
- 避免影响现有布局
- 逐页面测试效果
- 确保兼容性

---

## 📝 快速示例

### 启用后的典型页面

```xaml
<UserControl>
    <Border Style="{StaticResource CardStyle}">
        <StackPanel>
            <!-- 自动应用 BodyTextStyle -->
            <TextBlock Text="这是普通文本"/>
            
            <!-- 手动指定标题样式 -->
            <TextBlock Text="这是标题" 
                       Style="{StaticResource HeadingStyle}"/>
            
            <!-- 自动应用 StandardTextBoxStyle -->
            <TextBox/>
            
            <!-- 自动应用 SecondaryButtonStyle -->
            <Button Content="取消"/>
            
            <!-- 手动指定主要按钮样式 -->
            <Button Content="确定" 
                    Style="{StaticResource PrimaryButtonStyle}"/>
            
            <!-- 自动应用 CardListBoxStyle（卡片式选中效果） -->
            <ListBox ItemsSource="{Binding Items}" SelectedItem="{Binding SelectedItem}">
                <ListBox.ItemTemplate>
                    <DataTemplate>
                        <TextBlock Text="{Binding Name}"/>
                    </DataTemplate>
                </ListBox.ItemTemplate>
            </ListBox>
        </StackPanel>
    </Border>
</UserControl>
```

---

## 🔧 自定义配置

可以编辑 `ImplicitStyles.xaml` 修改默认样式：

```xml
<!-- 将默认按钮改为主要按钮样式 -->
<Style TargetType="Button" BasedOn="{StaticResource PrimaryButtonStyle}"/>

<!-- 将默认文字改为标题样式 -->
<Style TargetType="TextBlock" BasedOn="{StaticResource HeadingStyle}"/>
```

## 🔔 Toast通知使用

### 什么是Toast通知？

Toast通知是轻量级的桌面通知，在屏幕角落短暂显示后自动消失，不打断用户操作。

### 使用方法

> **💡 已启用全局using！** 所有C#文件无需添加using语句，可直接使用Toast通知。

```csharp
// 无需using语句！直接使用！

// 成功通知（绿色）
GlobalToast.Success("操作成功", "配置已保存");

// 警告通知（黄色）
GlobalToast.Warning("注意", "请检查配置参数");

// 错误通知（红色）
GlobalToast.Error("操作失败", "无法保存配置");

// 信息通知（蓝色）
GlobalToast.Info("提示", "这是一条信息");
```

### Toast vs StandardDialog

| 场景 | 使用 | 原因 |
|------|------|------|
| 成功提示 | GlobalToast.Success | 不打断操作流 |
| 警告提示 | GlobalToast.Warning | 轻量级提示 |
| 简单信息 | GlobalToast.Info | 快速显示 |
| 详细错误 | StandardDialog.ShowError | 需要查看详情 |
| 需要确认 | StandardDialog.ShowConfirm | 需要用户交互 |
| 需要输入 | StandardDialog.ShowInput | 需要用户输入 |

### Toast通知配置

Toast通知的样式由"通知设置 → 系统通知 → 通知样式"统一配置：
- 外观：圆角、阴影、边框、透明度
- 动画：类型、时长、缓动
- 布局：位置、大小、间距
